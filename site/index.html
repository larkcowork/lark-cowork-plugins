Bảo mật & Niềm tin

AI thao tác dữ liệu công ty — nên chúng tôi mặc định an toàn.

✓Dry-run mọi lệnh có side-effect — xem trước rồi mới chạy.
✓Mail luôn lưu draft tới khi bạn xác nhận — never auto-send.
✓Credential trong OS keychain, không file phẳng.
✓HTTP bearer-token constant-time; /health mở, / và /mcp khoá.
✓Audit log NDJSON — ghi lại mọi lệnh.
✓Token theo từng user — chỉ thấy dữ liệu mình có quyền, không rò chéo.

Minh bạch: AI có thể hallucinate và prompt injection tồn tại. Sau khi cấp quyền, agent hành động dưới danh tính bạn trong scope — khuyến nghị bắt đầu chế độ stdio/local cho Lãnh đạo trước khi mở HTTP/multi-tenant.

# 1. Build & cài binary (có lệnh mcp)
./scripts/setup-mcp.sh            # cài vào ~/bin/lark-cli
lark-cli mcp tools                # phải liệt kê 25 tools
# 2. Đăng nhập Lark một lần (OAuth → OS keychain)
lark-cli auth login
# 3. Khai báo vào claude_desktop_config.json rồi khởi động lại Claude Desktop
