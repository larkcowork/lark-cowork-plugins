# Lark IM Plugin

This plugin integrates Lark IM with Claude Code, providing tools to search, read, and send messages in Lark IM.

## Commands

- `/slack:summarize-channel <channel-name>` — Summarize recent activity in a Lark IM channel
- `/slack:find-discussions <topic>` — Find discussions about a specific topic across Lark IM channels
- `/slack:draft-announcement <topic>` — Draft a well-formatted Lark IM announcement and save it as a draft
- `/slack:standup` — Generate a standup update based on your recent Lark IM activity
- `/slack:channel-digest <channel1, channel2, ...>` — Get a digest of recent activity across multiple Lark IM channels

## Skills

- **slack-messaging** — Guidance for composing well-formatted Lark IM messages using mrkdwn syntax
- **slack-search** — Guidance for effectively searching Lark IM to find messages, files, channels, and people
